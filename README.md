## 🛠 How to Run

### 1. Install [Java JDK]

### 2. Use an IDE like **VS Code** or **Eclipse**

### 3. Clone or download this repository:
   ```bash
   git clone https://github.com/Sumathisree-A/Java-Programming-Knowledge-base
   ```

### 4. Compile the Java file:
    bash
    javac FileName.java
    
### 5. Run the program:
    bash
    java FileName
